# Data Directory

This directory contains course data stored as JSON files.

## Structure

Each course has its own JSON file:
- `cs-101.json` - Introduction to Computer Science
- `web-202.json` - Modern Web Development
- etc.

## Format

Course files include:
- Course metadata (title, description, instructor)
- Lessons array with components
- All edit history

## Important

- **DO NOT** commit sensitive course data
- These files are auto-generated by the CMS
- Backup before major changes
